About These Builds

These wheels are not maintained by the Open3D team. They are provided as-is for convenience while official Python 3.13+ support is pending. For production use, prefer official releases once available.

Compiling Open3D requires extensive CMake and C++ knowledge, plus serious build horsepower. I'm a seasoned C++ developer and I build these wheels in my spare time to bridge the gap until official support lands.

If these wheels save you hours of compilation time, consider supporting me by following on YouTube, X, and BlueSky.
